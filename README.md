# مدونة أحمد زين

مدونة شخصية عربية بتصميم عصري بسيط، مخصصة للنشر على GitHub Pages.

## التخصصات
- الفقه الحنبلي
- أصول الفقه
- مقاصد الشريعة
- علم الاجتماع
- علم النفس
- الاقتصاد الإسلامي
- الصيدلة والطب

## الملفات الموجودة
- `index.html` : الصفحة الرئيسية الجاهزة للنشر.
- `.nojekyll` : لتعطيل Jekyll حتى تُخدم الصفحة الثابتة كما هي.
- `LICENSE` : ترخيص الاستخدام.
- `assets/` : مجلد مخصص لأي صور أو ملفات إضافية مستقبلًا.

## طريقة الرفع إلى GitHub
1. أنشئ مستودعًا جديدًا على GitHub. إذا أردت أن تكون المدونة على الرابط الرئيسي لحسابك، سمّه `username.github.io`. [web:3]
2. ارفع كل ملفات هذا المشروع إلى جذر المستودع، وليس داخل مجلد فرعي. [web:3]
3. اذهب إلى `Settings` ثم `Pages`. [web:3]
4. من قسم `Build and deployment` اختر `Deploy from a branch`. [web:3]
5. اختر الفرع `main` والمجلد `/root` ثم احفظ. [web:3]
6. انتظر بضع دقائق حتى يظهر الموقع على رابط GitHub Pages، وقد يستغرق النشر عدة دقائق. [web:3]

## لماذا يوجد ملف .nojekyll ؟
أضفت ملف `.nojekyll` لأن الموقع هنا عبارة عن HTML ثابت، وهذا يمنع GitHub Pages من التعامل معه عبر Jekyll عند عدم الحاجة لذلك. [web:7]

## الخطوة التالية المقترحة
يمكنك لاحقًا تطوير هذا المشروع إلى:
- مدونة متعددة الصفحات.
- أرشيف مقالات وتصنيفات.
- صفحات مقالة منفصلة.
- ربط بنظام إدارة محتوى ثابت.
