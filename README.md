# سُلافة — مدونة فقه وفكر وقهوة

قالب مدونة عربية ثابتة، صُمم بروحٍ تراثية هادئة ولمسة حديثة. يعمل مباشرة على **GitHub Pages** ولا يحتاج إلى قاعدة بيانات أو بناء برمجي.

## المعاينة محليًا

افتح ملف `index.html` في متصفحك. ولأفضل تجربة، استخدم إضافة **Live Server** في VS Code إن أحببت.

## التعديل وإضافة مقال جديد

هذا الموقع ثابت، لذا لا توجد لوحة تحرير مخفية أو حسابات مستخدمين؛ أنت وحدك من يملك المستودع وتحرر المحتوى من ملفات المشروع على GitHub أو على جهازك.

1. انسخ الملف `articles/example-article.html` وأعطه اسمًا إنجليزيًا قصيرًا، مثل `articles/fiqh-al-taharah.html`.
2. عدّل العنوان، الوصف، التاريخ، التصنيف، ونص المقال داخل الملف الجديد.
3. أضف رابط المقال في `articles.html` تحت التخصص المناسب.
4. إن رغبت بظهوره في الصفحة الرئيسية، حدّث إحدى البطاقات داخل `index.html`.
5. ارفع التعديلات إلى GitHub. ستنشر GitHub Pages الموقع تلقائيًا خلال دقائق.

> مهم: في ملف المقال الموجود داخل مجلد `articles` تبقى ملفات التصميم والبرمجة مرتبطة هكذا: `../assets/css/style.css` و`../assets/js/main.js`.

## تخصيص سريع

- الاسم والشعار: ابحث عن `سُلافة` في ملفات HTML واستبدلهما.
- البريد: في `about.html` بدّل `your@email.com` ببريدك الحقيقي.
- الألوان: أعلى `assets/css/style.css` يحتوي المتغيرات اللونية. عدّل `--accent` للون البني الرئيس.
- الوضع الفاتح/الداكن: الزر في الشريط العلوي يحفظ اختيار الزائر تلقائيًا في متصفحه.

## النشر عبر GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub، ثم ارفع **محتويات هذا المجلد** (وليس المجلد نفسه).
2. من المستودع: `Settings` ← `Pages`.
3. اختر **Deploy from a branch**، ثم اختر فرع `main` ومجلد `/(root)`، ثم اضغط **Save**.
4. سيظهر رابط مدونتك في صفحة Pages بعد اكتمال النشر.

## البنية

```text
index.html                 الصفحة الرئيسية
articles.html              أرشيف المقالات
about.html                 عن المدونة والتواصل
articles/example-article.html  نموذج لمقال جديد
assets/css/style.css       التصميم والألوان المتجاوبة
assets/js/main.js          الوضع الداكن والسنة التلقائية
```
