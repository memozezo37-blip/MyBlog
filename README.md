# رِواق — مدونة أحمد زين العابدين

مدونة شخصية.

## نشر الموقع والتحكم الكامل

### GitHub Pages (الموصى به)
1. أنشئ حسابًا في [GitHub](https://github.com) إذا لم يكن لديك حساب.
2. أنشئ مستودعًا جديدًا، مثلاً `riwaq-blog`، واجعله **Public**.
3. ارفع كل ملفات هذا المجلد إلى المستودع.
4. من **Settings → Pages**، اختر النشر من فرع `main` ثم احفظ.
5. بعد دقائق سيظهر رابط مدونتك. من إعدادات Pages يمكنك ربط نطاقك الخاص لاحقًا.

## تعديل المحتوى

- غيّر اسمك، الوصف، والروابط الاجتماعية من ملف `index.html`.
- أنشئ مقالًا جديدًا بنسخ ملف `posts/maqasid.html`، ثم أضفه كبطاقة ضمن قسم «أحدث الكتابات» في `index.html`.
- حرر أي ملف من واجهة GitHub نفسها؛ كل حفظ سيحدّث الموقع تلقائيًا خلال دقيقة أو اثنتين.

> ملاحظة: نموذج الاشتراك في النسخة الحالية يعرض رسالة نجاح فقط. لتجميع البريد فعليًا، يمكن ربطه لاحقًا بخدمة مثل Buttondown أو Brevo.
