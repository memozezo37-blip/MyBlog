# مِداد وقهوة

واجهة عربية ثابتة لمدونة شخصية تجمع بين شغف القهوة والكتابة في العلوم الشرعية والإنسانية والصحية.

## المحتوى

- الفقه الحنبلي
- أصول الفقه
- مقاصد الشريعة
- علم الاجتماع
- علم النفس
- الاقتصاد الإسلامي
- الصيدلة والطب

## التشغيل محليًا

لا تحتاج الصفحة إلى أي تثبيت أو بناء. افتح ملف `index.html` مباشرةً في المتصفح.

## النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub وارفع هذه الملفات إلى الفرع `main`.
2. من إعدادات المستودع، افتح **Settings → Pages**.
3. اختر **Deploy from a branch**، ثم اختر الفرع `main` والمجلد `/(root)`.
4. احفظ الإعدادات؛ سيظهر رابط المدونة بعد اكتمال النشر.

## تخصيص سريع

- غيّر اسم المدونة والنصوص من `index.html`.
- عدّل الألوان من المتغيرات أعلى ملف `styles.css`.
- نموذج الاشتراك تجريبي في الواجهة فقط. اربطه بخدمة بريدية أو نموذج خارجي قبل استخدامه فعليًا.

## الملفات

```text
├── index.html   # الصفحة الرئيسية
├── styles.css   # التصميم المتجاوب
├── script.js    # القائمة التفاعلية والنموذج التجريبي
└── README.md    # تعليمات المشروع والنشر
```
