# مسائل

مدوّنة ثابتة (Static Site) بـ HTML و CSS و JS بسيطين، بلا أطر عمل ولا أدوات بناء.
تغطي سبعة مسارات: الفقه الحنبلي، أصول الفقه، مقاصد الشريعة، علم الاجتماع، علم النفس،
الاقتصاد الإسلامي، والصيدلة والطب.

## هيكل المشروع

```
masail-blog/
├── index.html          الصفحة الرئيسية (البطل، المسارات، أحدث المقالات، عن الكاتب)
├── css/
│   └── style.css        كل الأنماط: الألوان، الخطوط، التخطيط، الاستجابة
├── js/
│   └── main.js           قائمة الجوال + ظهور تدريجي عند التمرير
├── posts/
│   └── example-post.html  قالب صفحة مقال فردي — انسخه لكل مقال جديد
└── assets/                للصور والأيقونات المستقبلية
```

لا يوجد أي Build Step. الموقع HTML/CSS/JS خام، يعمل مباشرة بفتح `index.html`
أو برفعه كما هو على أي استضافة ثابتة.

## نظام التصميم

| العنصر | القيمة |
|---|---|
| خلفية أساسية | `#F6EFE3` كريمي دافئ |
| خلفية بطاقات | `#EFE4D2` |
| نص/خلفية داكنة | `#2A1A12` بُنّ محروق |
| لون تفاعلي أساسي | `#A85C32` نحاسي |
| تمييز ثانوي | `#C69A4C` ذهبي مطفأ |
| خط العناوين | Aref Ruqaa (Google Fonts) |
| خط النصوص | Tajawal (Google Fonts) |

العنصر البصري المميز هو الوردة (Rosette) في أعلى الصفحة: حلقات متداخلة على هيئة
حلقات فنجان القهوة، مرتّبة بتماثل دوراني ثماني — استعارة بصرية تجمع بين القهوة
والزخرفة الهندسية الإسلامية.

## إضافة مقال جديد

1. انسخ `posts/example-post.html` وأعد تسميته (مثلاً `posts/al-thamaniyya.html`).
2. عدّل: `<title>`، وصف meta، التصنيف (`eyebrow`)، العنوان، ومحتوى `article-body`.
3. أضف بطاقة جديدة للمقال في قسم «أحدث المقالات» بملف `index.html`، بنفس نمط
   البطاقات الموجودة (`.post-card`).
4. الروابط الداخلية في صفحة المقال نسبية (`../index.html`، `../css/style.css`) —
   احرص أن يبقى الملف داخل مجلد `posts/`.

## تعديل المسارات السبعة

كل مسار عبارة عن `<article class="track-card">` داخل قسم `#tracks` في `index.html`،
يحتوي أيقونة SVG وعنوانًا ووصفًا سطرًا واحدًا. لإضافة مسار ثامن أو تعديل الوصف،
عدّل داخل هذا القسم مباشرة.

## النشر على GitHub Pages

```bash
git init
git add .
git commit -m "إعداد أولي للمدونة"
git branch -M main
git remote add origin <رابط-المستودع>
git push -u origin main
```

ثم من إعدادات المستودع على GitHub:
`Settings → Pages → Source → Deploy from branch → main / (root)`

سيكون الموقع متاحًا على:
`https://<username>.github.io/<repo-name>/`

## ملاحظات تقنية

- الصفحة بالكامل RTL (`dir="rtl"`)، ومبنية بدون أي مكتبة أيقونات خارجية —
  كل الأيقونات SVG مكتوبة يدويًا داخل الصفحات.
- تُحترم إعدادات تقليل الحركة (`prefers-reduced-motion`) في كل من الدوران البطيء
  للوردة والظهور التدريجي للبطاقات.
- لا توجد أي اعتماديات JavaScript خارجية — `main.js` نحو 30 سطرًا بلا مكتبات.
