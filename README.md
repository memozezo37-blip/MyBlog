# مدونة أحمد – قهوة وعلوم شرعية وإنسانية

هذه مدونة شخصية تجمع بين حب القهوة، والكتابة في عدد من التخصصات، منها:

- الفقه الحنبلي  
- أصول الفقه  
- مقاصد الشريعة  
- علم الاجتماع  
- علم النفس  
- الاقتصاد الإسلامي  
- الصيدلة والطب  

## فكرة التصميم

- تصميم عصري بسيط (Minimal) بألوان مستوحاة من القهوة (درجات البني والبيج).
- واجهة انسيابية تركز على القراءة، مع إبراز التخصصات في بطاقات واضحة.
- قسم رئيسي (Hero) يربط بين القهوة والكتابة العلمية والإنسانية.

## طريقة التشغيل محليًا

1. **تحميل المشروع** أو نسخه:
   - إما عن طريق `git clone`:
     ```bash
     git clone https://github.com/USERNAME/REPO-NAME.git
     ```
   - أو تحميله كملف ZIP من GitHub.

2. **فتح الصفحة**:
   - افتح ملف `index.html` مباشرة في المتصفح (Chrome, Edge, Firefox…).

## رفع المشروع على GitHub Pages

1. أنشئ مستودع جديد على GitHub وضع فيه الملفات:
   - `index.html`
   - `style.css`
   - `README.md`
   - (وأي مجلدات أخرى مثل `assets/` إن احتجت)

2. ادفع التغييرات:
   ```bash
   git add .
   git commit -m "Initial blog layout"
