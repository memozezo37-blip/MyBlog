# مدونة القلم والقهوة

مدونة شخصية تعنى بالتقاطع بين العلوم الشرعية (الفقه الحنبلي، أصول الفقه، مقاصد الشريعة)، والعلوم الإنسانية (علم الاجتماع، علم النفس، الاقتصاد)، والعلوم التطبيقية (الصيدلة والطب).

## التقنيات المستخدمة
- HTML5: لبناء الهيكل الدلالي للموقع.
- CSS3: تنسيق عصري بنظام الـ Grid والـ Flexbox، مع متغيرات (Variables) للتحكم بالألوان.
- Vanilla JavaScript: لبرمجة زر التبديل بين الوضع الفاتح والداكن.

## طريقة الرفع على GitHub Pages
1. قم بإنشاء مستودع (Repository) جديد على GitHub.
2. ارفع هذه الملفات (`index.html`, `style.css`, `script.js`).
3. اذهب إلى إعدادات المستودع (Settings) > القائمة الجانبية (Pages).
4. في قسم **Build and deployment**، اختر `main` branch تحت قائمة Source، ثم اضغط Save.
5. سيظهر لك رابط مدونتك بعد دقائق.

## إضافة مقالات جديدة
بما أن هذا هيكل ثابت (Static)، تتم الإضافة بإنشاء ملفات HTML جديدة لكل مقال وربطها بالصفحة الرئيسية عبر وسم `<a>`.
