# مدينة الفكر

مدونة شخصية HTML بتصميم فقهي تراثي ولمسة عصرية انسيابية.

تجمع بين الفقه الحنبلي وأصول الفقه ومقاصد الشريعة، وعلوم الاجتماع والنفس والاقتصاد الإسلامي والصيدلة والطب، مع ركن للقهوة.

---

## التشغيل

1. افتح مجلد المشروع:
   ```
   madinat-al-fikr/
   ```
2. انقر مرتين على الملف **`index.html`** لفتحه في المتصفح.

لا تحتاج خادمًا ولا تثبيت حزم. المشروع يعمل محليًا مباشرة.

> **نصيحة:** إن رغبت في معاينة أنظف (خصوصًا مع بعض المتصفحات)، شغّل خادمًا محليًا بسيطًا من داخل المجلد:
>
> ```bash
> # Python
> python -m http.server 8080
> ```
>
> ثم افتح: `http://localhost:8080`

## أين تعدّل المحتوى؟

**الملف الوحيد المطلوب للتعديل اليومي:**

### `content.js`

| القسم | ماذا يفعل |
|--------|-----------|
| `SITE` | اسم المدونة، الاسم الظاهر، نصوص الرئيسية و«عن المدونة» وركن القهوة |
| `SPECIALTIES` | التخصصات السبعة (الاسم، الوصف، الأيقونة) |
| `ARTICLES` | المقالات |

### 1) بيانات المدونة (`SITE`)

```js
const SITE = {
  name: "مدينة الفكر",
  author: "اسمك هنا",
  // ...
};
```

- `hero` — عنوان ونص الصفحة الرئيسية والاقتباس  
- `about` — صفحة «عن المدونة»  
- `coffee` — نصوص ركن القهوة  

### 2) التخصصات (`SPECIALTIES`)

كل تخصص له:

| الحقل | المعنى |
|--------|--------|
| `id` | معرّف بالإنجليزية (يُستخدم في الروابط والفلاتر) — **لا تغيّره** بعد ربط مقالات به |
| `name` | الاسم العربي |
| `icon` | إيموجي |
| `category` | وسم قصير (شرعي / إنساني / …) |
| `short` | وصف البطاقة |
| `long` | الوصف الكامل في صفحة التخصص |

### 3) إضافة مقال جديد

انسخ كائنًا داخل `ARTICLES` وعدّل الحقول:

```js
{
  id: "my-article-slug",       // فريد، إنجليزي، بدون مسافات
  title: "عنوان المقال",
  excerpt: "ملخص يظهر في البطاقة.",
  specialty: "hanbali",        // يجب أن يطابق id تخصص موجود
  date: "2026-07-18",          // YYYY-MM-DD
  readTime: 6,                 // دقائق القراءة التقريبية
  content: `
    <p>فقرة...</p>
    <h2>عنوان فرعي</h2>
    <blockquote>اقتباس</blockquote>
    <ul>
      <li>نقطة</li>
    </ul>
  `,
}
```

**قيم `specialty` المتاحة حاليًا:**

| `specialty` | التخصص |
|-------------|--------|
| `hanbali` | الفقه الحنبلي |
| `usul` | أصول الفقه |
| `maqasid` | مقاصد الشريعة |
| `sociology` | علم الاجتماع |
| `psychology` | علم النفس |
| `islamic-econ` | الاقتصاد الإسلامي |
| `pharmacy-med` | الصيدلة والطب |

### HTML المسموح داخل `content`

- `<p>` فقرات  
- `<h2>` / `<h3>` عناوين فرعية  
- `<blockquote>` اقتباسات  
- `<ul>` / `<ol>` / `<li>` قوائم  
- `<strong>` / `<em>` تأكيد  

بعد الحفظ، أعد تحميل الصفحة في المتصفح (`F5`).

## الصفحات والتنقّل

| الرابط | الصفحة |
|--------|--------|
| `#/` | الرئيسية |
| `#/specialties` | كل التخصصات |
| `#/specialty/hanbali` | تخصص واحد (مثال) |
| `#/articles` | كل المقالات |
| `#/articles/usul` | مقالات تخصص معيّن |
| `#/article/fiqh-soul-and-method` | مقال واحد |
| `#/coffee` | ركن القهوة |
| `#/about` | عن المدونة |

التنقّل داخلي (SPA): الصفحة لا تُعاد تحميلها بالكامل عند الانتقال بين الأقسام.

## الوضع الفاتح والداكن

- الزر في أعلى الشريط (شمس / قمر).  
- الاختيار يُحفظ في `localStorage` تحت المفتاح: `madinah-theme`.  
- إن لم يختر الزائر شيئًا، يُحترم تفضيل نظام التشغيل.

## التخصيص الاختياري

| الهدف | الملف |
|--------|--------|
| الألوان، الخطوط، المسافات | `assets/css/style.css` (متغيرات `:root` و`[data-theme="dark"]`) |
| سلوك التنقّل أو الفلاتر | `assets/js/app.js` |
| هيكل الصفحة أو القائمة | `index.html` |

### ألوان سريعة (في `style.css`)

```css
:root {
  --accent: #1b4332;   /* الأخضر الفقهي */
  --gold: #b8923a;     /* الذهب */
  --bg: #f6f1e7;       /* خلفية الرقّ */
}
```

## المزايا الحالية

- تصميم RTL عربي كامل  
- تراثي (خط أميري، زخارف، أخضر وذهب) × عصري (بطاقات، انتقالات، جوال)  
- تبديل إضاءة فاتح / داكن  
- سبع تخصصات + أرشيف مقالات مع تصفية  
- ركن القهوة  
- محتوى مركزي سهل التعديل من `content.js`  
- يعمل بدون إنترنت بعد التحميل الأول للخطوط (أو بدون خطوط جوجل إن قُطعت الشبكة)

---

## ملاحظات

1. **لا تحذف** `id` تخصص عليه مقالات منشورة دون تحديث المقالات.  
2. **`id` المقال** يصبح جزءًا من الرابط — اختره بعناية.  
3. المقالات تُرتَّب تلقائيًا من الأحدث إلى الأقدم حسب `date`.  
4. للرفع على استضافة ثابتة (GitHub Pages، Netlify، أي استضافة ملفات): ارفع مجلد `madinat-al-fikr` كاملًا كما هو.
