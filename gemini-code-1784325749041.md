# مدونة أحمد زين العابدين الشخصية

مدونة ثابتة (Static Blog) تجمع بين التصميم التراثي الأنيق والانسيابية الحديثة، مبنية خصيصاً للاستضافة على GitHub Pages.

## الميزات
- **تصميم مزدوج:** إضاءة فاتحة (ورق وقهوة) وإضاءة داكنة مريحة للعين.
- **تحديث سلس:** لا حاجة للتعامل مع كود HTML لإضافة مقال. جميع المقالات تضاف عبر ملف `posts.json`.
- **خطوط مدروسة:** خط "أميري" للترويسات لإضفاء الطابع الفقهي التاريخي، و"القاهرة" للمتن لتسهيل القراءة السريعة.

## كيفية إضافة مقال جديد (دليلك الشخصي)
لإضافة مقال، قم بفتح ملف `posts.json` الموجود في المستودع. أضف كائن JSON جديداً في أعلى القائمة بهذا الشكل:

```json
{
    "id": 3,
    "title": "عنوان مقالك هنا",
    "date": "التاريخ",
    "category_en": "رمز القسم الإنجليزي للمبرمج",
    "category_ar": "اسم القسم كما يظهر للقارئ",
    "content": "نص المقال يكتب هنا."
}
